# FieldFlow

Offline-first humanitarian workflow platform. Deployed at https://fieldflow-tau.vercel.app

## Tech Stack
- Next.js 16 PWA, React 19, TypeScript, Tailwind CSS
- IndexedDB offline storage, operation-based sync protocol
- AWS DynamoDB + Cognito, Vercel deployment
- DeepSeek AI Workflow Architect

Visit https://fieldflow-tau.vercel.app and use any demo account to explore all roles.
